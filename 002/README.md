

یک مدل یادگیری ماشین بسازید که الگوی اعداد زیر را یاد بگیرد:
 
5, 10, 15, 20, 25, 30, ...
 
به طوری که اگر شماره جمله (مثلاً جمله اول، دوم، سوم) را به مدل بدهید، مقدار آن جمله را پیش‌بینی کند.

مثال:
- ورودی 1 → خروجی 5
- ورودی 2 → خروجی 10
- ورودی 3 → خروجی 15
- ورودی 4 → خروجی 20

مدل باید بتواند برای ورودی‌های جدید (مثل جمله دهم) مقدار درست را پیش‌بینی کند.



Build a machine learning model that learns the following number pattern:

5, 10, 15, 20, 25, 30, ...

The model should take the position of a term (1st, 2nd, 3rd, ...) as input and predict the value of that term.

Example:
- Input 1 → Output 5
- Input 2 → Output 10
- Input 3 → Output 15
- Input 4 → Output 20

The model should be able to predict the correct output for new inputs (e.g., 10th term → 50).
