

- ورودی: 1 → خروجی: 1
- ورودی: 2 → خروجی: 3
- ورودی: 3 → خروجی: 5
- ورودی: 4 → خروجی: 7


یعنی مدل باید بفهمه که **خروجی = (ورودی × 2) - 1**


یک مدل یادگیری ماشین بسازید که رابطه بین اعداد ورودی و خروجی را یاد بگیرد. به عنوان مثال:
- اگر ورودی 1 باشد، خروجی 1
- اگر ورودی 2 باشد، خروجی 3
- اگر ورودی 3 باشد، خروجی 5
- اگر ورودی 4 باشد، خروجی 7

مدل باید بتواند برای ورودی‌های جدید (مثل 10) خروجی درست را پیش‌بینی کند.


Build a machine learning model that learns the relationship between input and output numbers. For example:
- Input 1 → Output 1
- Input 2 → Output 3
- Input 3 → Output 5
- Input 4 → Output 7

The model should be able to predict the correct output for new inputs (e.g., input 10).
